# My-WGS-Project

## プロジェクト概要

小学3・4年生向け漢字テストWebアプリ（シングルHTMLファイル）。
ブラウザで動作し、サーバー不要。子どもが楽しく漢字を練習できることを目標としている。

## ファイル構成

```
kanji-quiz.html   メインアプリ（HTML/CSS/JS すべて一体）
README.md
```

## 現在の実装状態

### 機能
- **設定画面**：学期選択（1〜3学期）＋テスト種類選択
- **読みテスト**：漢字→ひらがな入力、自動採点
- **書きテスト**：ひらがな→手書きキャンバス（マウス・タッチ対応）、自己採点
- **結果画面**：スター評価（0〜3）、スコア表示、全問の正誤復習リスト
- 進捗バー・フィードバックアニメーション
- レスポンシブ対応（スマホ・タブレット）
